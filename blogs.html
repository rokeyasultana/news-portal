What's the deference between var,const & let?

Var variable is functional scope.Let variable is block scope.Const variable is block scope.It can be updated and re-declared into the scope.It can be updated but cannot be re-declared into the scope.It cannot be updated or re-declared into the scope.

Difference between regular functions and arrow functions

In regular function, Arguments keywords can be used to access the arguments of which passed to function. Arrow functions do not have an arguments binding.Regular functions are constructible, they can be called using the new keyword. However, arrow functions can never be used as constructor functions. Hence, they can never be invoked with the new keyword.In regular function, function gets hoisting at topIn arrow function, function get hoisted where you define.
